# 🧮 Simple Calculator — CLI App

A command-line calculator built with Python 3. Supports addition, subtraction, multiplication, division, and modulus operations with full input validation and error handling.

---

## 📸 Preview

```
========================================
        SIMPLE CALCULATOR
========================================

Select an operation:
  1. Addition       (+)
  2. Subtraction    (-)
  3. Multiplication (*)
  4. Division       (/)
  5. Modulus        (%)
  6. Exit
----------------------------------------
Enter your choice (1-6): 4
Enter first number:  10
Enter second number: 2

  Result: 10.0 / 2.0 = 5.0
```

---

## ⚙️ Features

- Addition, Subtraction, Multiplication, Division, Modulus
- Looping menu — keeps running until you choose Exit
- Input validation — handles letters and symbols gracefully
- Exception handling — catches ZeroDivisionError and ValueError
- Clean, readable output

---

## 🚀 How to Run

**Step 1 — Make sure Python is installed:**
```bash
python --version
```

**Step 2 — Clone this repository:**
```bash
git clone https://github.com/rksingh-108/calculator-cli.git
cd calculator-cli
```

**Step 3 — Run the app:**
```bash
python calculator.py
```
