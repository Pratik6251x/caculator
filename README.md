🔢 Simple Calculator — CLI App

A command-line calculator built with Python 3 that performs basic arithmetic operations with strong input validation and error handling.

🖥️ Preview
========================================
        SIMPLE CALCULATOR
========================================

Select an operation:

1) Addition        (+)
2) Subtraction     (-)
3) Multiplication  (*)
4) Division        (/)
5) Modulus         (%)
6) Exit

----------------------------------------
Enter your choice (1-6): 4

Enter first number : 10
Enter second number: 2

Result → 10.0 / 2.0 = 5.0
🧰 Features

✔ Addition
✔ Subtraction
✔ Multiplication
✔ Division
✔ Modulus

Extra capabilities:

🔄 Continuous running menu until Exit is selected

🧾 Input validation (handles invalid inputs)

⛔ Protection against division by zero

🧼 Clean CLI output formatting

🪶 Lightweight — no external libraries

🚀 How to Run
1️⃣ Check Python installation
python --version
2️⃣ Clone the repository
git clone https://github.com/Pratikcli/calculator-cli.git
3️⃣ Move to the project folder
cd calculator-cli
4️⃣ Run the program
python calculator.py
